# Social_Analytics/

This folder contains engagement summary reports for each social media platform.

## Files
- `Twitter_Summary.md` — Twitter/X weekly engagement metrics
- `Facebook_Instagram_Summary.md` — Facebook Page + Instagram business account metrics
- `LinkedIn_Summary.md` — LinkedIn post performance

## Updated By
- `SKILL_Twitter_Draft` — updates `Twitter_Summary.md` after generating analytics
- `SKILL_Facebook_Instagram` — updates `Facebook_Instagram_Summary.md`
- `SKILL_LinkedIn_Draft` — updates `LinkedIn_Summary.md`
- `SKILL_Weekly_CEO_Briefing` — reads all summaries to include in CEO Briefing

## Used By
- `SKILL_Weekly_CEO_Briefing` — reads these files to include social performance in the Sunday briefing
- `Dashboard.md` — Social Media Status table is populated from these files

## Metrics Tracked
| Platform | Metrics |
|----------|---------|
| Twitter/X | Impressions, engagements, link clicks, follower change |
| Facebook | Page reach, post engagement, DMs received |
| Instagram | Reach, engagement rate, profile visits, DMs |
| LinkedIn | Post impressions, reactions, comments, profile views |
